## food-receipt
나라별 대표 음식 레시피 앱

### 주요 기능
- 각 나라별 대표 음식들의 리스트 확인
- category에서 해당 국가를 클릭하면 대표 음식들을 확인 할 수 있음
- 음식 이미지를 클릭하면 사용 재료와 조리 레시피가 나옴
- search 부분에서 음식 검색 기능 가능
 
### 사용 기술
React-Native,
JavaScript / TypeScript,
React Navigation

### 프로젝트 목적
각 나라의 대표음식을 알리고 해당 레시피를 통해
직접 사용자가 체험할 수 있도록
