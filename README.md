# 🍽️ Food Recipe App

<p align="center">
국가별 대표 음식 레시피 앱
url - https://snack.expo.dev/@no15sunga/food-receipt-app
</p>

---

## 📌 프로젝트 개요

본 프로젝트는 다양한 국가의 대표 음식을 탐색하고,  
각 음식의 재료와 레시피를 확인할 수 있는 모바일 애플리케이션이다.

사용자는 국가를 선택하여 대표 음식 목록을 확인하고,  
각 음식에 대한 상세 정보를 직관적으로 탐색할 수 있다.

---

## 🚀 주요 기능

- 🌍 국가별 대표 음식 조회  
- 🖼️ 음식 이미지 기반 리스트 제공  
- 📖 음식 클릭 시 재료 및 레시피 상세 보기  
- 🔍 검색 기능을 통한 빠른 음식 탐색  

---

## 🧩 기능 설명

- **국가 선택 기능**  
  각 나라를 선택하면 해당 국가의 대표 음식 리스트를 확인 가능  

- **음식 상세 정보 조회**  
  음식 이미지를 클릭하면 재료와 레시피를 상세하게 제공  

- **검색 기능**  
  음식 이름을 검색하여 원하는 레시피를 빠르게 탐색 가능
